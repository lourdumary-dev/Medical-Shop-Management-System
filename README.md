# Medical Shop Management System

## Project Description

Medical Shop Management System is a Flask-based web application developed to manage medicines in a medical shop. It allows the administrator to add, update, search, delete medicines and generate bills efficiently using a MySQL database.

## Features

- Secure Login
- Dashboard
- Add Medicine
- View Medicines
- Search Medicine
- Update Medicine
- Delete Medicine
- Generate Bill
- Sales History
- MySQL Database Integration

## Technologies Used

- Python
- Flask
- HTML
- CSS
- MySQL
- mysql-connector-python

## Installation

### Clone Repository

```bash
git clone https://github.com/lourdumary-dev/Medical-Shop-Management-System.git
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Create MySQL Database

Create a database named:

```
medical_shop
```

Import the required tables.

### Run the Project

```bash
python app.py
```

### Open Browser

```
http://127.0.0.1:5000
```
